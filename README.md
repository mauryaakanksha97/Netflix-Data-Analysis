# Netflix Data Analysis 

## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on Netflix’s content catalog using Python.  
The goal is to uncover insights about content distribution, growth trends, duration patterns, and country-wise availability.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Source: Netflix Movies and TV Shows dataset
- Format: CSV
- Rows: ~19,000 titles
- 
## Key Analysis Performed
- Movies vs TV Shows distribution
- Content growth trend over the years
- Average duration of movies
- Country-wise content distribution
- Missing values analysis (Unknown directors/cast)
- Genre and release year trends

## Key Insights
- Netflix content is dominated by **Movies** compared to TV Shows.
- Rapid growth in content additions after **2015**, reflecting platform expansion.
- The **United States and India** are top content-producing countries.
- A noticeable portion of titles have **unknown directors or cast**, indicating metadata gaps.

## Business Recommendations
- Increase investment in **TV Shows and web series** to improve user retention.
- Expand **localized content** production for emerging markets.
- Improve **metadata completeness** to enhance search and recommendation systems.
